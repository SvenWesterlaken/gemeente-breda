# Android Naming

## Layout

```xml
activityName_abbreviatedType_nameOfObject

Example: main_TV_firstName (for a TextView in the activity_main layout meant for first names)
```

Most common types:
- TextView: TV
- ImageView: IV
- EditText: ET
- ListView: LV
- Button: BTN
- Divider: D

## Variables

```java
private TextView firstName;
```
